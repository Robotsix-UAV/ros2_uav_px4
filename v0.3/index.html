ROS2 UAV PX4 Documentation

Overview

Welcome to the ROS2 UAV PX4 documentation. This package uses the uav_cpp and the PX4 ROS2 Interface libraries to provide a high-level interface to the PX4 Autopilot.

The main entry point for this package is the modes_runner node. This node provides additional mode that are registered to the PX4 Autopilot for offboard control.

This documentation provides guidance to users. Developers can find additional information at the Developer Documentation page.
