README

ROS2 UAV PX4

Overview

ROS2 UAV PX4 is a ROS 2 package designed to facilitate communication between a companion computer and the PX4 autopilot software using the PX4 ROS 2 Interface Library and the UAV CPP library.
Currently, this repository supports the main branch of PX4-Autopilot. Future updates will ensure compatibility with stable releases (1.15 and later).

Documentation

Comprehensive documentation can be found at ROS2 UAV PX4.

Support

If you encounter any issues or have feature requests, please open an Issue on GitHub.

For personal support, you can contact the maintainer at damien@robotsix.net.

Contributions to the project are welcome. If you would like to contribute, please submit a Pull Request on GitHub.
